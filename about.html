CourseGuide: What you need to know about my courses

Course guides in HTML, PDF, and ePub by Peter Smith

This site is something of an experiment on my part; it grew out of a desire to find a better way to provide my course material to students.

The site is largely built using the most excellent bookdown package by Yihui Xie. I have also drawn on the template of the bookdown.com site to bootstrap this site into existence.

Some tech info

This site is hosted as static site using Amazon Simple Storage Service (S3) cloud storage. Eventually, I will move the source of the books onto GitHub to allow editing by other folk.
